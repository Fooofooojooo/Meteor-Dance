# Meteor Dance
A space arcade game where a meteor shower is fast approaching your spacecraft. Don't panic just yet — the stars are dancing, and if you follow their rhythm, they'll light the path to safety.

# How to install
- Click the Download button on the itch.io page
- Once downloaded, find the zip file in your Downloads folder
- Right click the zip file and select "Extract All"
- Choose where you want to extract it and click Extract
- Open the extracted folder
- Double click Meteor Dance.exe to launch the game
- If Windows shows a security warning, click "More info" then "Run anyway"

# How to play
- Use the arrow keys to move your spacecraft left and right
- Watch the stars, when one lights up, a meteor is incoming at that position
- Survive as long as possible and rack up your high score
- The longer you survive, the more meteors fall

# Features
- Warning system — stars light up before a meteor arrives so you have time to react
- Difficulty scaling — meteor showers get more intense as your score climbs
- High score saving — your best run is saved between sessions
- Pulsing star background — stars gently animate, making warnings harder to spot
